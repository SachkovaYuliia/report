Тема: ФУНКЦІОНАЛЬНЕ ЗАСТОСУВАННЯ JAVASCRIPT У HTML-ДОКУМЕНТІ. ВИКОРИСТАННЯ МАСИВІВ У JS-СЦЕНАРІЯХ. РЕАЛІЗАЦІЯ ПРОГРАМ ЗАСОБАМИ МОВИ JAVASCRIPT

Мета: придбати практичні навички роботи з масивами у js-сценаріях. Реалізація програм засобами мови JAVASCRIPT

Завдання

1.Створити одновимірний масив, кількість елементів якого задана користувачем. Знайти суму елементів з парними індексами, максимальний елемент та його індекс, мінімальний елемент та його індекс серед елементів з непарними індексами.

2. Упорядкувати масив в порядку зростання, Надрукувати вхідний та вихідний масив.

Застосувати функції.

Виконання

Завдання

Динамічне меню

Сформувати багаторівневе меню (горизонтальне та вертикальне), пункти меню взяти із статичного масиву. При натисканні на визначений пункт меню має завантажуватися відповідна HTML-сторінка. Меню має бути присутнім на всіх сторінках.

Виконання

Висновки

Під час виконання завдання отримано практичні навички роботи з масивами у js-сценаріях. Опрацьовано методи роботи з масивами. Засобами мови JAVASCRIPT реалізовано багаторівневе динамічне меню.
